# Snooplink - Professional Social Media Scheduler PWA

Snooplink is a modern, high-performance Progressive Web App (PWA) designed for seamless social media content scheduling. Built with React and Firebase, it provides a 100% cost-free solution for managing multiple social media platforms including Facebook, Instagram, and TikTok.

## 🚀 Key Features

- **Multi-Platform Scheduling**: Plan and schedule posts for Facebook Pages, Instagram Business, and TikTok Business from a single dashboard.
- **Zero-Cost Storage**: Utilizes Firestore for cost-free media handling through Base64 encoding (100% free hosting usage).
- **Professional Developer Config**: Integrated API credentials management for official Graph API and TikTok API connections.
- **SaaS Architecture**: Modular folder structure with dedicated pages for Dashboard, Scheduler, Media Library, Accounts, and Analytics.
- **Custom Firestore Auth**: Secure, database-level authentication system without external SDK dependencies.
- **Premium UI/UX**: Stunning Light Mode interface with rich aesthetics, modern typography (Outfit), and smooth animations.
- **PWA Support**: Fully installable on mobile and desktop with offline capabilities.

## 🛠️ Technology Stack

- **Frontend**: React.js, Vite
- **Styling**: Vanilla CSS (Custom Design System)
- **Database**: Firebase Firestore
- **Deployment**: Firebase Hosting
- **Icons**: Lucide React, FontAwesome
- **Animations**: Framer Motion

## 📦 Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/PWA-Snooplink.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Firebase**:
   Update `src/firebase.js` with your Firebase project credentials.

4. **Run locally**:
   ```bash
   npm run dev
   ```

5. **Deploy to Firebase**:
   ```bash
   npm run build
   firebase deploy
   ```
